NoteLang

A Programming Language for Music Creation

Introduction

NoteLang is a domain-specific programming language designed for creating music through code. It combines traditional programming constructs with musical notation, allowing you to write programs that generate MIDI music files.

Installation

1. Clone the repository:

git clone https://github.com/ColmenarAlex/NoteLang.git

2. Install dependencies:

pip install textx midiutil

3. Run your first NoteLang program:

python NoteLang.py examples/melody.notelang

Example Programs

Simple Scale

score {
    tempo = 120
    
    : 4C4 
    : 4D4 
    : 4E4 
    : 4F4
    : 4G4 
    : 4A4
    : 4B4 
    : 4C5
}

Arpeggio with Block

score {
    tempo = 120
    
    block arpeggio {
        : 8C4
        : 8E4
        : 8G4
    }
    
    play arpeggio
    : 4R
    play arpeggio
}

Variable Octaves

score {
    octave = 4
    while octave < 6 {
        : 4Coctave
        : 4Eoctave
        : 4Goctave
        octave += 1
    }
}

Language Specification

Basic Syntax

Program Structure: Every program must start with score { and end with }

Note Format: [duration][note][octave][accidental?]

  • Duration: 1 (whole), 2 (half), 4 (quarter), 8 (eighth)
  • Note: A through G
  • Octave: number (4-6) or variable
  • Accidental: # (sharp) or b (flat) - optional

Rest: [duration]R

Variables:

  • Assignment: variable = value
  • Modification: variable += value

Blocks:

block name {
    : notes...
}
play name

Control Flow:

while condition {
    commands...
}
